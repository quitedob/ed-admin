## 🎯 系统概述

题库管理系统是一个功能强大的教育内容管理平台，支持多种题型的创建、编辑、分析和智能管理。

### 核心功能

- **📝 题目管理**: 支持单选、多选、填空、判断、编程、简答等多种题型
- **🏷️ 智能标签**: 层级化标签系统，支持分类管理和快速筛选
- **🤖 AI助手**: 智能题目生成、优化和解题步骤生成
- **📹 视频讲解**: 支持本地视频上传和外部平台视频集成
- **📊 数据分析**: 全面的题目使用统计和趋势分析
- **♿ 无障碍支持**: 完整的键盘导航和屏幕阅读器支持

### 系统要求

- 现代浏览器（Chrome 90+, Firefox 88+, Safari 14+, Edge 90+）
- 屏幕分辨率最低 1024x768
- JavaScript 必须启用
- 网络连接稳定

## 📝 题目管理

### 创建新题目

1. 点击顶部「新建题目」按钮
2. 选择题型：
   - **单选题**: 一个正确答案
   - **多选题**: 多个正确答案
   - **填空题**: 文本填空
   - **判断题**: 是非判断
   - **编程题**: 代码编写
   - **简答题**: 开放性回答

3. 填写题目基本信息：
   - **题目标题**: 简明扼要的题目名称
   - **题目内容**: 使用富文本编辑器
   - **难度等级**: 简单/中等/困难
   - **分值**: 题目分数

4. 设置选项（适用于选择题）：
   - 添加/删除选项
   - 标记正确答案
   - 调整选项顺序

5. 完善高级设置：
   - **标签分类**: 选择相关知识点标签
   - **题目解析**: 详细解答过程
   - **解题提示**: 给学生的提示信息

### 编辑题目

1. 在题目列表中找到要编辑的题目
2. 点击「操作」按钮，选择「编辑」
3. 修改题目内容
4. 保存更改

### 复制题目

1. 选择要复制的题目
2. 点击「操作」>「复制题目」
3. 系统创建题目副本，可进一步编辑

### 删除题目

1. 选择要删除的题目
2. 点击「操作」>「删除」
3. 确认删除操作

⚠️ **注意**: 删除操作不可撤销，请谨慎操作。

## 🔍 高级筛选

### 筛选选项

- **关键词搜索**: 在标题和内容中搜索
- **题型筛选**: 按题目类型过滤
- **难度筛选**: 简单/中等/困难
- **学科分类**: 计算机科学/数学/物理/化学等
- **标签筛选**: 多选标签组合筛选
- **日期范围**: 按创建时间筛选
- **排序方式**: 创建时间/更新时间/使用次数/难度

### 搜索技巧

- 使用空格分隔多个关键词
- 使用引号进行精确搜索："JavaScript闭包"
- 使用减号排除关键词：算法 -排序
- 结合多个筛选条件提高精度

## 🤖 AI助手功能

### 启动AI助手

点击顶部「AI助手」按钮，打开智能助手面板。

### AI功能列表

#### 1. 从文本生成题目
1. 点击「从文本生成题目」
2. 粘贴或输入文本内容
3. 选择题目类型和难度
4. 设置生成数量
5. 点击「生成题目」

#### 2. 生成相似题目
1. 选择一个已有题目
2. 点击「生成相似题目」
3. AI将基于原题目创建变体
4. 调整参数后生成

#### 3. 优化题目
1. 选择要优化的题目
2. 点击「优化现有题目」
3. AI分析题目质量
4. 提供改进建议

#### 4. 生成解题步骤
1. 选择题目
2. 点击「生成解题步骤」
3. AI提供详细解析
4. 包含知识点讲解

### AI对话功能

在聊天区域可以直接与AI助手对话：

- **问题示例**:
  - "帮我创建一个关于React Hooks的单选题"
  - "这道题目有什么可以改进的地方？"
  - "为这道算法题生成详细的解题步骤"

- **技巧**:
  - 描述越详细，生成结果越准确
  - 可以指定难度和知识点
  - 支持多轮对话调整

## 🏷️ 标签管理

### 打开标签管理

点击顶部「标签管理」按钮。

### 标签分类

系统支持层级化标签结构：

```
知识点
├── 基础概念
├── 进阶知识
└── 高级应用

难度等级
├── 入门级
├── 进阶级
└── 专家级

学科领域
├── 计算机科学
├── 数学
├── 物理
└── 化学
```

### 创建标签

1. 选择或创建标签分类
2. 点击「新建标签」
3. 填写标签信息：
   - **标签名称**: 简洁明了
   - **所属分类**: 选择上级分类
   - **标签颜色**: 设置显示颜色
   - **描述信息**: 标签用途说明

### 标签使用

- **添加标签**: 在题目编辑时选择相关标签
- **批量操作**: 可以批量添加或修改标签
- **标签搜索**: 使用标签快速筛选题目

## 📹 视频讲解

### 上传本地视频

1. 在题目详情页面点击「视频讲解」标签
2. 点击「添加视频讲解」
3. 选择「上传本地视频」
4. 拖拽或选择视频文件
5. 填写视频信息：
   - **视频标题**: 清晰的标题
   - **描述信息**: 视频内容说明

### 添加外部视频

1. 选择视频平台（Bilibili、YouTube等）
2. 粘贴视频链接
3. 系统自动获取视频信息
4. 补充说明文字

### 视频管理

- **预览播放**: 在线预览视频内容
- **编辑信息**: 修改视频标题和描述
- **删除视频**: 移除不需要的视频
- **排序调整**: 调整视频播放顺序

## 📊 数据分析

### 访问分析页面

点击顶部「数据分析」按钮，进入分析仪表板。

### 统计概览

- **总题目数**: 系统中的题目总量
- **总使用次数**: 题目被使用的总次数
- **平均正确率**: 所有题目的平均答题正确率
- **AI生成题目**: AI助手创建的题目数量

### 图表分析

#### 1. 题目类型分布
- 饼图显示各题型占比
- 支持时间范围筛选

#### 2. 难度分布
- 环形图显示难度分布
- 识别难度设置是否合理

#### 3. 使用趋势
- 折线图显示使用趋势
- 支持不同时间维度
- 预测未来使用情况

#### 4. 学科分布
- 柱状图显示学科分布
- 识别热门学科领域

#### 5. 标签热度
- 显示最常用标签
- 了解知识点覆盖情况

### 数据导出

1. 设置筛选条件
2. 选择导出格式（Excel/CSV）
3. 点击「导出数据」
4. 下载分析报告

## ❓ 常见问题

### Q: 如何批量导入题目？
A: 点击顶部「批量导入」按钮，支持Excel、CSV格式，请按照模板格式准备数据。

### Q: AI生成的题目质量如何保证？
A: AI生成后请人工审核和调整，确保题目准确性和合适性。

### Q: 如何备份数据？
A: 使用导出功能定期备份，或联系系统管理员进行数据备份。

### Q: 支持哪些视频格式？
A: 本地上传支持MP4、AVI、MOV格式，最大100MB；外部链接支持主流视频平台。

### Q: 如何提高题目搜索效率？
A: 1) 合理设置标签 2) 规范题目标题 3) 使用高级筛选组合条件。

### Q: 数据分析多久更新一次？
A: 实时数据，图表会自动更新，也可以手动刷新获取最新数据。

### Q: 如何设置题目访问权限？
A: 在系统设置中配置角色权限，或联系管理员设置访问控制。

### Q: 支持移动端访问吗？
A: 系统响应式设计，支持平板设备，手机端建议使用横屏模式。
